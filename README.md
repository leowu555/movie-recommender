# movie-recommender

A full-stack movie recommendation platform built with Django, React, PostgreSQL, Scikit-learn, and TMDB API.

## Tech Stack
- Frontend: React
- Backend: Django REST Framework
- Database: PostgreSQL
- Machine Learning: Scikit-learn
- External API: TMDB API

## MVP Features
- User authentication
- Movie search
- Movie details page
- Ratings system
- Watchlist
- Personalized recommendations
